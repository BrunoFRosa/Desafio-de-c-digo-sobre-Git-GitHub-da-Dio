cd + (pasta) = Entrar na pasta especificada
cd .. = Voltar um nivel
Ctrl + l = Limpar Términal
del (pasta) = Apaga os arquivos
git init = Inicia um repositório
git config --global user.email "seu e-mail" = Passa seu e-mail para o Git
git config --global user.name "seu nome" = Passa seu nome para o Git
git add + nome do aqruivo = Adiciona para fazer um commit
git add* = Adiciona tudo para fazer um commit
git commit -m = Faz oo commit dos arquivos
git status = Mostra o estado dos arquivos
git config --list = Mostra os dados configurados no Git
git config --global --unset user.(nome/e-mail) = Altera ou remove os dados
git remote add origin.(link do github) = Aponta o repositório para uma origem remota
git remote -v = Lista
git push origin master = Empurra um commit para o repositório remoto
git pull origin master = Puxa o código do GitHub
ls = Listar
ls -a = Mostra pastas ocultas
mkdir = Criar uma pasta
rmdir (pasta) /s /q = Apaga tudo
tab = auto completar
Windows + cmd = Prompt de comando
